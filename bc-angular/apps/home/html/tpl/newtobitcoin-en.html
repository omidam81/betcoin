Online Bitcoin Wallet Introduction and Usage Guide

What is Bitcoin?

Bitcoin is a type of digital currency in which encryption techniques are used to regulate the generation of units of currency and verify the transfer of funds, operating independently of a central bank.

For more information, visit bitcoin.org or wikipedia.

Create A Bitcoin Wallet

Blockchain.info

Blockchain.info is one of the most reliable and powerful online Bitcoin wallets available – for novice and experienced users alike. It requires no software installation or lengthy registration processes, taking just seconds to get up and running.

First, visit the Blockchain.info wallet homepage. From here, click on the green Create My Free Wallet button.

All you need to register is your email address and a suitable password of at least 10 characters in length. It’s important that you enter a secure password, so the site provides automated advice as you enter if a dictionary word or insecure combination is detected.

When you’ve registered, you will receive a confirmation email containing information about the My Wallet service and a link to activate your account. After logging in to your wallet, you should see the Wallet Homepage with a newly created address that’s ready to use for Bitcoin transactions.

When you wish to access your account in the future, you can use the login link supplied in your Welcome Email or click on the Wallet link in the Blockchain homepage header - it contains an ‘identifier’ which is used to access your wallet (along with your password).

This identifier should be automatically saved in your browser (as a cookie), although if it disappears, refer to the Welcome Email to access it.

Receive Money

Select the "Receive" link to view all of your bitcoin addresses.

You can select the "Actions" button in the menu to sign a message, make a backup or perform other wallet functions

Monitor Transactions

Select "My Transactions" tab to view your complete transaction history. When a new transaction is made, it will automatically be placed in this area

Send Money

Select the "Send" link to send money. You should use the recommended fee for your transaction, to make sure it reaches the bitcoin network and is confirmed promptly

Sign a Message

To sign a message, first select the "Receive" link, then select "Actions" button next to the address you want to sign a message with. A popup will open asking you to enter a message to sign. Enter the desired message and select "Sign". Then you can copy/paste the resulting signed message.

Electrum

Electrum is a Bitcoin wallet built for speed and convenience. A streamlined design and rich feature set make this software a popular choice for beginners and experts alike.

Downloading Electrum

Visit the Electrum downloads page and save the Electrum installation file to your PC. You need to select the correct download for your operating system! Make sure to check the MD5 signature of the download to avoid downloading software that's been tampered with

Starting Up Electrum

Once you've downloaded and started Electrum for the first time you'll be asked to create a wallet. Select any server from the list and click ok.

A seed will be generated for you. You must keep this seed in a safe place, it will let you restore your wallet in the future.

It is recommended to set a strong password to encrypt your wallet.

Now you can start using Electrum to send and receive Bitcoins.

Receive Money

New addresses are generated automatically when your receiving addresses are used.

Locate a receiving address by first clicking on the "Receive" tab. Double-click the first "Receiving" address, then copy it to the clipboard by selecting the "Copy to clipboard" menu item.

Monitor Transactions

The "History" tab lists all current and pending transactions. An entry appears soon after a transaction is made into a wallet address. Initially, this transaction will be marked as "pending".

Send Money

Click on the "Send" tab to send bitcoins. Make sure you use the recommended fee to ensure your bitcoins are credited as quickly as possible to the receiver.

Sign a Message

Select the "Tools" menu and click "Sign A Message".

You will need to fill in the bitcoin address to sign your message with and the message to be signed, and then select "Sign". You will be asked for your wallet password. Copy/paste the resulting signed message.
